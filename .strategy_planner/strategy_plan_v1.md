## 一、当前策略状态总览

### 1.1 核心逻辑

**策略**：开盘缺口回补（Opening Gap Fill）

| 要素 | 描述 |
|------|------|
| **开仓逻辑** | 当日开盘价相对前日收盘产生显著跳空 → 反向交易，等待缺口回补 |
| **方向判定** | 向上跳空 → 做空；向下跳空 → 做多 |
| **出场机制** | 固定时间出场（~45 分钟为主）+ 止损 2×ATR + 止盈 3×ATR |
| **信号评分** | 8 因子加权评分 → 综合置信度，≥70% 才入场 |
| **品种池** | 9 个精选品种：CU, TA, P, RB, AG, SC, I, MA, SA |
| **信号过滤** | 跳过周一 + 跳过 7 月/12 月 |
| **风险控制** | 2% 单笔风险上限 |

## 二、已知问题深度分析

### 问题 1：出场机制形同虚设（151/154 = 98% 时间出场）

当前止损 2×ATR 和止盈 3×ATR 触发次数极少。原因分析：

- **日线回测的局限**：用当日 K 线的最高/最低判断止损/止盈是否触发，但这模拟的是"全天任何时刻"，而非"开盘后 45 分钟内"。如果 ATR 基于日线计算，2×ATR 在 45 分钟内被触发的概率远高于日线回测的估算
- **真实分钟级场景**：止损的实际触发率会比回测高得多 → 当前 Sharpe 3.58 可能高估
- 这意味着：**当前策略实际上是一个"固定时间持有"策略**，而非有风险管理手段的策略

### 问题 2：滑点是隐形成本黑洞（21% 的利润被吃掉）

- 净利润 +24.6 万，滑点 -5.2 万 → 滑点吃掉 21% 利润
- 减少非必要交易是降低滑点最直接的方式（已通过提高阈值实现）
- 但 154 笔已是最精简配置，进一步提阈值会损失太多信号

### 问题 3：做空严重拖后腿

中国市场期货的长期 contango 结构 + 政策底 + 经济周期特征，使得做空天然处于劣势。缺口回补策略的均值回归逻辑在做空方向上受到结构性阻力。

### 问题 4：数据质量瓶颈

分钟级数据仅覆盖 2-3 周，远不足以做统计推断。日线回测用收盘价模拟出场价格，系统性地美化了回测结果。

### 问题 5：高缺口区间的反直觉现象

> 缺口 > 2% 的胜率最高（68.9%），而中等缺口 1-2% 胜率反而较低

可能解释：
- 大缺口是情绪极度恐慌/贪婪的产物 → 均值回归力更强
- 现在设置了 max_gap=3%，已过滤掉极端缺口 → 剩余的大缺口恰好是最适合回补的区间
- 或者样本量太小（大缺口本身就不多），存在幸存者偏差

## 三、改进方向提案（按优先级排序）

### 🔥 优先级 1：非对称多空处理

**为什么这是最高优先级**

做多贡献 88% 利润，做空几乎不赚钱。即使完全不做空，理论上也能保持 +21.6 万的收益，同时省掉做空交易的手续费和滑点。这是投入产出比最高的改进方向。

**理论基础**

1. **中国市场结构性特征**：商品期货长期 contango（远月升水），做空承担 roll yield 损失
2. **均值回归的非对称性**：恐慌性下跌后的反弹（做多回补）比贪婪性上涨后的回落（做空回补）更加迅速和可靠。行为金融学中，损失厌恶导致下跌后的修复动力更强
3. **政策/库存底**：中国商品有政策和库存双重底部支撑，做空的下行空间有限但上行空间未知

**改进方案**（三个递进选项）

| 方案 | 描述 | 风险 |
|------|------|------|
| A. 保守 | 做空阈值提高到 75%，做多保持 70% | 损失部分空头机会 |
| B. 中性 | 做空权重打折（所有因子得分 ×0.7），等效阈值更高 | 保留少量高质量空头 |
| C. 激进 | 完全不做空（long-only） | 放弃空头分散化，但可能不亏 |

**预期效果**
- 方案 A/B：胜率提升 3-5%，Sharpe 提升 0.5-1.0，总收益可能持平或小幅提升
- 方案 C：收益减少 ~3 万但 Sharpe 可能提升（减少低质量交易）

**实验设计**
1. 在现有 run_v3.py 中增加 `direction_filter` 参数
2. 对比 A/B/C 三个方案在 9 品种 + 70% 阈值下的表现
3. 重点看：收益变化、Sharpe、最大回撤、交易数变化
4. 如果方案 B 既保留了空头的分散化，又显著提升了效率 → 最优

---

### 🔥 优先级 2：动态出场优化（时间出场 → 条件出场）

**为什么重要**

当前 98% 的出场是时间出场，意味着策略几乎没有主动风险管理。固定 45 分钟持有是简化假设，不是最优参数。

**理论基础**

1. **缺口回补的时间分布不均匀**：有些缺口 10 分钟就回补了，有些 60 分钟还没回补。固定 45 分钟 = 经常错过最佳出场点
2. **盘中量能衰减**：开盘前 30 分钟是量能高峰，之后逐步衰减。如果缺口在量能高峰期未回补，后期回补概率下降
3. **ATR 在日内需要重新校准**：日内波动率分布呈 U 型（开盘高、午盘低、尾盘高）

**改进方案**

| 方案 | 描述 |
|------|------|
| A. 多时间窗口 | 增加 15/30/60/90 分钟出场选项，根据缺口幅度动态选择 |
| B. 条件出场 | 缺口回补达到 X% 即出场（如回补 70% 就平仓）+ 剩余仓位继续持有或止损 |
| C. 量能追踪 | 如果持仓 30 分钟后 Volume Profile 显示量能持续萎缩 → 提前出场 |
| D. trailing stop | 入场后用移动止损替代固定止损/止盈，让利润跑 |

**预期效果**
- 方案 A（多时间窗口）：Sharpe 提升 0.2-0.5，主要是通过减少不必要持仓降低回撤
- 方案 B（部分止盈）：胜率显著提升，因为很多交易"接近回补但没收全"
- 方案 D（移动止损）：盈亏比改善，抓住少数大利润交易

**实验设计**
1. **前置条件**：需要足够的分钟级数据（至少 1 年），目前数据不足
2. **短期**：在日线回测框架中模拟不同出场时间的效果（如用当日高点/低点的时序位置估算）
3. **中期**：积累 3-6 个月的 5 分钟数据后，跑完整的日内出场优化
4. **对比指标**：不同出场方案下的 Sharpe、胜率、盈亏比、最大回撤

### 🔥 优先级 3：品种级个性化参数

**为什么重要**

AG 用 44 笔交易只赚 1.6 万，效率极低。这意味着统一参数对不同品种的效果差距巨大。简单方案：直接剔除 AG，但更优雅的方案是让参数适配品种特性。

**理论基础**

1. **波动率结构不同**：CU（铜）和 SA（纯碱）的日内波动率特性完全不同，2×ATR 止损对 SA 太宽、对 CU 太窄
2. **缺口回补速度不同**：流动性好的品种（如 RB 螺纹钢）回补更快，需要更短的持有时间
3. **季节性模式不同**：农产品（P 棕榈油）的季节性与工业品（CU 铜）完全不同
4. **AG 的特殊性**：沪银受外盘 COMEX 影响大，跳空可能是外盘隔夜行情的滞后反映，而非纯粹的均值回归机会。这意味着 AG 的缺口"信息含量"不同于纯内盘品种

**改进方案**

| 方案 | 描述 | 复杂度 |
|------|------|--------|
| A. 剔除 AG | 品种池从 9→8，简单直接 | 低 |
| B. 分组参数 | 金属组/化工组/农产品组各自优化阈值和止损 | 中 |
| C. 逐品种优化 | 每个品种独立回测最优的 gap_threshold/stop_loss/hold_minutes | 高 |
| D. AG 特殊处理 | AG 增加外盘联动因子，仅在外盘同向跳空时更高置信度 | 高 |

**预期效果**
- 方案 A：立即见效，交易数 110 笔，收益几乎不变（~+23 万），Sharpe 可能提升
- 方案 B：Sharpe 提升 0.3-0.8，因为各品种的出场参数更合理
- 方案 C：最大提升，但需要足够样本量（每品种至少 50 笔），回测可能过拟合
- 方案 D：最有深度，但需要外盘数据接入

**实验设计**
1. 先用方案 A 跑一次对比回测（删 AG），确认效果
2. 按品类分组（有色：CU+AG、化工：TA+MA+SA、黑色：RB+I、农产品：P+SC），跑分组优化
3. 对比统一参数 vs 分组参数 vs 剔除 AG 的效果
4. 如果分组效果明显好于统一，再考虑进一步到逐品种优化

### 优先级 4：分钟级数据积累与真实回测

**为什么重要**

当前所有回测都是日线级别的，用当日收盘价模拟出场，这系统性地美化了回测结果。真实的分钟级回测会暴露：
- 止损在盘中触发的实际频率（会比日线估算高得多）
- 滑点在不同时间段的实际分布
- 缺口实际回补的时间分布

**计划**
1. **每周自动下载**：周一至五收盘后自动下载当日 5 分钟 K 线
2. **数据积累目标**：3 个月后（~60 个交易日）开始初步分析；6 个月后做正式分钟级回测
3. **短期验证**：用现有 2-3 周数据做"烟雾测试"——验证代码逻辑是否正确，不做统计推断

**预期效果**
- 分钟级回测的 Sharpe 预计低于日线回测（更真实）
- 但能发现日线回测看不到的优化机会（如最优出场时间、止损宽度）

### 优先级 5：缺口上下文增强（新因子开发）

**为什么重要**

当前因子体系对"缺口为什么会发生"关注不够。同样 1.5% 的跳空，如果是因为前一天夜盘突发新闻 vs 正常波动，回补概率完全不同。

**改进方向**

| 新因子 | 描述 | 预期贡献 |
|--------|------|----------|
| Overnight Range | 夜间（昨晚收盘→今早开盘）的价格区间 → 判断跳空是否"已经走了一大段" | 过滤假缺口 |
| Pre-Gap Trend | 缺口前 3 天的趋势方向 → 趋势加速跳空 vs 趋势反转跳空 | 区分顺势/逆势 |
| Gap Cluster | 近 5 日是否已有多个缺口 → 多缺口后回补概率更低（市场失衡） | 已有 TimeDecay，可增强 |
| Relative Gap Position | 缺口相对于近期波动区间的百分位 | 补充 VolatilityRegime |
| Volume-Weighted Gap | 缺口时的成交量 × 缺口幅度 → 加权冲击力 | 比单独看量能更有效 |

**验证方法**
1. 新因子单独回测：固定其他 7 个因子权重，观察加入新因子后 Sharpe/胜率变化
2. 因子相关性分析：确保新因子与已有因子不高度相关（冗余因子无价值）
3. 如果新因子贡献增量信息（如 Sharpe +0.2），纳入评分体系

## 五、风险提示

1. **过拟合风险**：154 笔交易的样本量很小。在这么小的样本上反复优化参数，很容易过拟合历史行情。建议：每次改进至少用 walk-forward validation 验证
2. **市场结构变迁**：2005-2026 跨了 21 年，市场微观结构发生了巨大变化（交易机制、流动性、参与者结构）。早期的策略表现可能不代表未来。建议：做滚动窗口回测，看策略在不同时期的稳定性
3. **做空完全舍弃的隐藏风险**：如果完全不做空，在熊市年份（如 2008, 2015 的大跌行情）会完全踏空。可能有更好的方案是保留高质量空头信号
4. **分钟级真实回测可能打破高 Sharpe 幻觉**：日线回测的 Sharpe 3.58 大概率高估，实际分钟级回测可能在 1.5-2.5 区间

以下是我认为值得深入讨论的问题，供后续交流：
