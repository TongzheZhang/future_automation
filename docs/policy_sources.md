# 政策数据源跟踪清单

> 本文件记录所有监控的政策数据源，以及各数据源对期货品种的影响映射。
> 持续更新迭代。

---

## 一、核心政策源（P0-P1 优先级）

### 1. 国务院 / 中共中央
- **来源**: http://www.gov.cn/zhengce/zhengceku/
- **影响范围**: 全市场系统性影响
- **关键文件类型**: 政府工作报告、中央一号文件、国务院常务会议、国办发文件
- **期货映射**: 
  - 农业政策 → 豆粕、玉米、棉花、棕榈油
  - 地产/基建政策 → 螺纹钢、铁矿石、焦炭
  - 能源政策 → 原油、甲醇、动力煤
  - 环保政策 → 黑色、有色、化工

### 2. 国家发展和改革委员会 (NDRC)
- **来源**: https://www.ndrc.gov.cn/xxgk/
- **影响范围**: 产能调控、价格干预、重大项目审批
- **关键文件类型**: 产业政策、价格调控政策、储备投放/收储通知
- **期货映射**:
  - 钢铁产能置换 → 螺纹钢、热卷、铁矿石
  - 煤炭保供/限价 → 焦炭、焦煤、动力煤
  - 粮食收储/投放 → 玉米、大豆、棉花
  - 油价调控 → 原油、化工品
  - 储备铜/铝投放 → 铜、铝

### 3. 中国人民银行 (PBOC)
- **来源**: http://www.pbc.gov.cn/zhengcehuobisi/
- **影响范围**: 流动性、汇率、利率
- **关键文件类型**: 货币政策执行报告、LPR调整、降准降息、外汇干预
- **期货映射**:
  - 宽松政策 → 贵金属（黄金、白银）↑、商品整体↑
  - 人民币贬值 → 进口依存度高的品种↑（大豆、铜、铁矿石、原油）
  - 人民币升值 → 出口型品种有利（部分化工品）

### 4. 工业和信息化部 (MIIT)
- **来源**: https://www.miit.gov.cn/zwgk/zcwj/
- **影响范围**: 工业行业管理、产能政策、技术标准
- **关键文件类型**: 行业准入条件、产能置换方案、技术标准更新
- **期货映射**:
  - 钢铁/有色行业规范 → 螺纹钢、铁矿石、铜、铝、镍
  - 新能源汽车政策 → 铜、镍、锂、钴（相关产业链）
  - 光伏政策 → 白银（银浆）、铝

### 5. 商务部 (MOFCOM)
- **来源**: https://www.mofcom.gov.cn/article/b/
- **影响范围**: 进出口政策、关税、贸易救济
- **关键文件类型**: 进口配额、反倾销/反补贴调查、关税调整
- **期货映射**:
  - 大豆进口关税/配额 → 豆粕、豆油
  - 棉花进口配额 → 棉花
  - 铜/铝废碎料进口政策 → 铜、铝
  - 棕榈油进口政策 → 棕榈油

### 6. 农业农村部 (MARA)
- **来源**: http://www.moa.gov.cn/gk/zcfg/
- **影响范围**: 农业生产、种植结构、疫病防控
- **关键文件类型**: 种植意向、收储政策、疫病通报、进出口政策
- **期货映射**:
  - 大豆/玉米种植补贴 → 豆粕、玉米
  - 猪瘟/禽流感通报 → 豆粕（饲料需求）、玉米
  - 棉花目标价格 → 棉花

### 7. 生态环境部 (MEE)
- **来源**: https://www.mee.gov.cn/xxgk2018/xxgk/xxgk15/
- **影响范围**: 环保限产、排放标准
- **关键文件类型**: 环保督察通报、排放标准修订、限产通知
- **期货映射**:
  - 环保限产 → 螺纹钢、焦炭、铜、铝、化工品
  - 碳排放政策 → 铝（高耗能）、钢

### 8. 国家能源局 (NEA)
- **来源**: https://www.nea.gov.cn/xxgk/
- **影响范围**: 能源生产、新能源、油气
- **关键文件类型**: 能源规划、保供方案、新能源装机目标
- **期货映射**:
  - 煤炭保供 → 焦炭、焦煤、动力煤
  - 油气勘探开发 → 原油、天然气
  - 新能源装机 → 铜、镍、白银、铝

### 9. 海关总署 (GACC)
- **来源**: http://www.customs.gov.cn/customs/302249/
- **影响范围**: 进出口数据、关税执行
- **关键文件类型**: 月度进出口数据、关税调整
- **期货映射**:
  - 大豆进口量 → 豆粕
  - 铁矿石进口量 → 铁矿石
  - 铜/铝进出口 → 铜、铝
  - 原油进口 → 原油

## 二、辅助信息源（P2 优先级）

### 行业媒体与数据机构
- **我的钢铁网 (Mysteel)**: 黑色系日报、周度数据
- **隆众资讯**: 能源化工数据
- **卓创资讯**: 大宗商品数据
- **粮油商务网**: 农产品数据
- **上海有色网 (SMM)**: 有色数据
- **中国棉花信息网**: 棉花专项数据

### 国际数据源
- **USDA**: 全球农产品供需报告（WASDE）
- **OPEC**: 原油月度报告
- **IEA**: 国际能源署报告
- **LME/COMEX**: 有色库存与持仓
- **CFTC**: 持仓报告（COT）
