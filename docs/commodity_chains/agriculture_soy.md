# 农产品-大豆压榨产业链分析

> 版本: v1.0 | 最后更新: 2026-05-25

## 产业链结构

```
进口大豆(A) / 国产大豆
    ↓
压榨厂
    ↓
豆粕(M) + 豆油(Y)
    ↓         ↓
饲料        食用油
    ↓
生猪(LH) / 禽类 / 水产
```

## 核心政策敏感点

### 进口政策
- **大豆进口关税**：中美贸易摩擦的核心品种
- **进口配额**：转基因大豆进口许可
- **国储拍卖**：临储大豆/菜籽油拍卖

### 农业政策
- **大豆振兴计划**：国产大豆种植面积补贴
- **中央一号文件**：每年农业政策纲领

### 养殖政策
- **非洲猪瘟**：影响豆粕需求的核心变量
- **生猪产能调控**：能繁母猪存栏目标

## 供需特点

- **进口依存度 > 85%**：中国大豆严重依赖进口（巴西、美国、阿根廷）
- **压榨利润决定进口节奏**：压榨利润高 → 买船积极 → 未来到港增加
- **季节性**：南美大豆 3-5 月上市，北美 9-11 月上市

## 交易机会类型

1. **进口政策突变**：关税调整 → 做多/做空豆粕
2. **南美天气**：拉尼娜/干旱 → 做多豆粕
3. **养殖周期**：生猪存栏恢复 → 做多豆粕
4. **压榨利润套利**：利润极端 → 反向操作
