# 铜产业链分析

> 版本: v1.0 | 最后更新: 2026-05-25

## 产业链结构

```
铜矿开采（智利、秘鲁、刚果金）
    ↓
铜精矿
    ↓
冶炼厂 → 精炼铜(CU)
    ↓
铜杆/铜管/铜板
    ↓
电力 / 空调 / 新能源汽车 / 电子
```

## 核心政策敏感点

### 能源与环保政策
- **电力成本**：电解铜高耗能，电力政策影响显著
- **环保督察**：冶炼厂限产

### 新能源政策
- **光伏/风电/电动车**：铜需求增长的核心驱动力
- **电网投资**：特高压、智能电网

### 进出口政策
- **废铜进口**：废铜进口标准调整影响供应
- **铜精矿进口**：依赖海外，汇率影响显著

### 货币政策
- **美元定价**：铜以美元计价，人民币汇率影响进口成本
- **美联储政策**：影响全球流动性与工业金属需求预期

## 供需特点

- **矿端瓶颈**：铜矿开发周期长（10年+），短期供应弹性低
- **冶炼产能相对过剩**：中国冶炼产能大，加工费(TC/RC)是核心指标
- **新能源需求**：电动车用铜量是传统车的 3-4 倍

## 交易机会类型

1. **矿端扰动**：智利罢工/刚果金政策 → 做多铜
2. **美元走弱**：美联储降息 → 做多铜
3. **新能源政策**：电动车补贴/电网投资 → 做多铜
4. **冶炼限产**：环保/电力紧张 → 做多铜（冶炼端收缩）
