# 黑色系产业链分析

> 版本: v1.0 | 最后更新: 2026-05-25

## 产业链结构

```
铁矿石(I) + 焦炭(J) + 焦煤(JM)
    ↓
高炉冶炼 → 生铁
    ↓
转炉/电炉 → 粗钢
    ↓
连铸连轧 → 螺纹钢(RB) / 热卷(HC)
    ↓
地产 / 基建 / 制造业
```

## 核心政策敏感点

### 供给侧改革
- **产能置换**：新建产能需淘汰旧产能，影响长期供应
- **环保限产**：秋冬季环保检查，短期供应冲击
- **粗钢压减**：年度产量控制目标

### 地产政策
- 螺纹钢 60%+ 需求来自地产
- 关注：住建部政策、保交楼进度、房企融资政策

### 基建刺激
- 专项债发行节奏
- 发改委重大项目审批

## 利润传导

```
钢厂利润 = 钢材价格 - (铁矿石成本 + 焦炭成本 + 其他)
```

- **钢厂高利润** → 增产 → 原料需求↑ → 铁矿石/焦炭↑
- **钢厂亏损** → 减产 → 钢材供应↓ → 螺纹钢相对抗跌

## 交易机会类型

1. **政策限产**：环保政策突袭 → 做多螺纹钢，做空铁矿石（钢厂利润扩大）
2. **地产刺激**：放松调控 → 做多螺纹钢
3. **供应释放**：产能集中投产 → 做空螺纹钢
4. **利润套利**：钢厂利润极端低位 → 多螺纹空铁矿
